# RCB IPL Data Analysis 🏏

## 📊 Project Overview
This project analyzes Royal Challengers Bangalore (RCB) IPL data using SQL to generate insights on team performance and player statistics.

## 🔍 Key Insights
- Match performance trends
- Top players and run scorers
- Win/loss analysis
- Venue-based performance

## 🛠️ Technologies Used
- SQL (MySQL / PostgreSQL)

## 📂 Project Structure
- Dataset files
- SQL queries
- Analysis results

## ▶️ How to Run
1. Import dataset into your database
2. Execute SQL queries
3. Analyze results
